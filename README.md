# Early Detection of Sepsis using Machine Learning

## Project Overview

Sepsis is a life-threatening medical condition caused by the body's extreme response to infection. Early detection is critical for improving patient survival rates.

This project presents a Machine Learning-based web application that predicts the risk of Sepsis based on patient clinical features.

The system uses trained ML models integrated with a Flask web application to provide real-time predictions.

---

## Features

- Predicts sepsis risk using trained machine learning models
- Supports multiple disease conditions
- User-friendly web interface
- Real-time prediction results
- History tracking of predictions

## Installation

Clone the repository


git clone https://github.com/Furqaan333/EARLY-DETECTION-OF-SEPSIS-.git


Navigate to project folder


cd EARLY-DETECTION-OF-SEPSIS-


Install dependencies


pip install -r requirements.txt


Run the application


python app.py


Open in browser


http://127.0.0.1:5000
